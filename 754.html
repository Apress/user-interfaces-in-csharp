Errata

Following is a list of known errata in the first edition of User Interfaces in C#.

Chapter 2

p. 39    The highlighted tip paragraph indicates that the DebuggerStepThroughAttribute is applied by Visual Studio .NET. This is only the case in VB .NET, not in C#.

Chapter 3

p. 71    Access keys are are triggered by the Alt key, not the Ctrl key as indicated in the text.

p. 80    The description beside "Image..." refers to Figure 3-8. It should refer to Figure 3-9 instead.

Chapter 4

p. 91    Access keys are are triggered by the Alt key, not the Ctrl key as indicated in the UseMnemonic section.

p. 103    In first code sample, udAge.Vale should be udAge.Value

p. 103    In second code sample, tackbar.SmallChange should be barVolume.SmallChange

p. 142    In Table 4-16, "a*b matches aab or just b", not "aab or just a".

Chapter 5

p. 145    In Table 5-1, describing Opacity, "system" should be plural.

p. 154    In second paragraph "ScrollableControl" is misspelled as "ScrollablControl".

p. 154    The final two sentences of the last paragraph claim you can can create a dialog form that is modal only to one other form the user can use access other Windows in your application. This is not correct, although you could show forms on multiple different threads to achieve this result.

p. 160    SecondaryForms should be an ArrayList, not a Hashtable (because keys are not used in this collection).

p. 164    In second paragraph. "... which you must be set to System ..." should be "... which you must set to System ..."

p. 196    In the last paragraph. "... and it also appears as value ..." should be "and is also appears as a value"

p. 198    In the second paragraph there is an extra comma in the first sentence (between "form" and "and").

Chapter 6

p. 217    At bottom of page, TreeMode newNode = new TreeNode(); should be TreeNode newNode = new TreeNode();

p. 241    The line of code "private tree as System.Windows.Forms.TreeView;" should be "private System.Windows.Forms.TreeView tree;"

Chapter 9

p. 368    e.Value += ((decimal)e.Value).ToString("c"); should be e.Value = ((decimal)e.Value).ToString("c");
